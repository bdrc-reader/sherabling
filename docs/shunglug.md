# འདིར་བཞུགས་པའི་གཞུང་ལུགས་ཁག་གི་ངོ་སྤྲོད།

དཔེ་ཀློག་སྒེའུ་ཁུང་འདིའི་ནང་ཞེ་ཆེན་དགོན་གྱི་བསླབ་པ་སློབ་གཉེར་བ་རྣམས་ཀྱིས་སློབ་གཉེར་གཙོ་བོ་གནང་སའི་སྐྱེས་ཆེན་རྣམས་ཀྱི་གསུང་རབ་འགའ་བཀོད་ཡོད། མདུན་ངོས་སུ་འགྲེལ་བཤད་བཀོད་པ་ལྟར་དཔེ་ཀློག་སྒེའུ་ཁུང་གི་སྡེ་ཚན་རེ་རེའི་ནང་ཕེབས་ཏེ། དོན་གཉེར་ཅན་ཚོས་ཕབ་ལེན་དང་བལྟ་ཀློག དག་སྒྲིག ཁ་གསབ་བཅས་བྱ་ཆོག་པ་ཡིན།
